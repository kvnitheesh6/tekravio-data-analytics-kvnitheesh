# Tekravio Data Analytics Internship Assignment

## Overview

This repository contains my submission for the Tekravio Data Analytics Internship Assignment.

The project demonstrates an end-to-end data analytics workflow using a real-world dataset. It covers data cleaning, exploratory data analysis, SQL analysis, dashboard creation, statistical analysis, and machine learning to generate meaningful business insights.

## Objectives

- Clean and preprocess raw data
- Perform Exploratory Data Analysis (EDA)
- Answer business questions using SQL
- Build an interactive Power BI dashboard
- Apply statistical analysis
- Build and evaluate machine learning models
- Present business recommendations

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Power BI
- Scikit-learn
- SciPy
